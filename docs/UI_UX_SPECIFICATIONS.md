# UI/UX SPECIFICATIONS

**Project**: cPort Credit Union Translation Tool  
**Purpose**: Comprehensive Screen-by-Screen UI/UX Design Specifications  
**Last Updated**: February 2026  
**Design Philosophy**: Ultra-Modern Minimalism with Warmth

---

## DESIGN VISION

### User Experience Goals

1. **Zero Learning Curve** - Staff should be productive within 5 minutes
2. **Calm Interface** - Financial transactions require focus, not distraction
3. **Confidence in Every Interaction** - Users must trust the technology
4. **Speed Without Sacrifice** - Fast workflows that don't cut corners
5. **Inclusive by Design** - Works for everyone, regardless of technical ability

### Emotional Design Targets

| User | Feeling We Create |
|------|-------------------|
| Greeter | Confident, helpful, in control |
| Teller | Efficient, focused, accurate |
| Consultor | Professional, thorough, caring |
| Customer | Understood, respected, at ease |

### Interactions

1. **Auto-focus** on email field on load
2. **Tab navigation** between fields
3. **Enter key** submits form from any field
4. **Loading state** shows spinner in button, disables inputs
5. **Error state** shakes form, red border on invalid field, error message below
6. **Success** redirects to appropriate dashboard based on role

### Accessibility

- Focus ring visible on all interactive elements
- Error messages announced to screen readers
- Password field has toggle visibility button
- Form labels properly associated

## MICRO-INTERACTIONS

### Button Feedback

- Scale to 0.98 on press
- Return with slight overshoot (spring ease)
- Ripple effect from touch point

### Success Celebration

- Subtle confetti burst for completed sessions
- Green checkmark with scale animation
- Haptic feedback (if available)

### Language Selection

- Card lifts on hover (translateY -4px)
- Border glows with language color
- Selected state has persistent glow
